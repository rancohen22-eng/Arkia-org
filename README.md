# arkia-org — מערכת עץ ארגוני של ארקיע

מערכת ווב לבניית **עץ ארגוני שבונה את עצמו**: כל מנהל ממלא רק את הכפופים הישירים
לו, ולכל כפוף שהוא מנהל נוצר **קישור-קסם אישי** לשליחה בוואטסאפ — וכך העץ מתפשט
מלמעלה למטה עד העלים, בלי שאף אחד ימפה את כל המבנה. עברית, RTL.

הוצא כמודול עצמאי ממערכת התמחיר `arkia-pricing`.

## הרצה מקומית
```bash
python -m venv .venv && . .venv/bin/activate   # אופציונלי
pip install -r requirements.txt
# משתמש לדוגמה (או צור users.txt בפורמט user:pass):
export ARKIA_USERS="ranc:1234"
python -m uvicorn app.main:app --port 8020
```
פתח http://localhost:8020 (או `run.bat` בווינדוס).

## מבנה
- `app/main.py` — כל ה-routes: התחברות, מסך האדמין, דף המילוי הציבורי, וה-API.
- `app/services/org.py` — שכבת הנתונים של העץ (יצירה/הוספה/עריכה/מחיקה/סטטיסטיקות).
- `app/services/org_export.py` — ייצוא דף HTML עצמאי לשיתוף.
- `app/templates/` — `base.html`, `login.html`, `org_admin.html` (התרשים), `org_fill.html` (דף מנהל).
- `app/static/css/app.css` — סגנון בסיס.
- `app/db.py` — SQLite (`data/org.db`, לא במאגר). `app/auth.py` — התחברות session-cookie.

## איך משתמשים
1. נכנסים למסך האדמין ומוסיפים **ראש מחלקה** (למשל סמנכ"ל כספים) → נוצר קישור.
2. לכל צומת אפשר **➕ להוסיף כפוף בעצמך**, **↥ להשחיל תפקיד באמצע**, **✏️ לערוך**,
   או **📲 לשלוח קישור** בוואטסאפ למנהל שימלא את הענף שלו.
3. תצוגה **אופקית/אנכית**, התאמה למסך, **הדפסה/PDF**, ו**ייצוא דף HTML לשיתוף**.
4. מנהל שמקבל קישור פותח `/org/fill/<token>` (בלי סיסמה) וממלא רק את הכפופים שלו.

## אבטחה
הגישה לדף המילוי מבוססת אך ורק על ה-`token` הסודי שבכתובת; הטוקן מגביל כל מנהל
לעריכת הכפופים הישירים שלו בלבד. מסכי האדמין וה-API שלהם דורשים התחברות.
